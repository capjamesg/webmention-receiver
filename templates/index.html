What are webmentions?

The webmention specification provides a way in which websites can send interactions to each other such as likes, replies, and bookmarks.

Many sites owned by IndieWeb community members (including this one!) support receiving webmentions.

By using webmentions, site owners can send comments to each other and retain control over their comment. This is not possible on traditional social media platforms where the platform usually owns all comments and content you produce.

How can I learn more about webmentions?

If you want to get started with webmentions, check out the IndieWeb Webmention wiki page. That page contains guidance on the basics of webmentions, how they are used, and how you can start adding webmentions to your own site.

Can I host my own webmention endpoint?

Yes! You can host your own webmention endpoint on your own server. You can deploy a version of this webmention endpoint by going to the project GitHub page and following the instructions.
